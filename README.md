# ls_base

## 项目简介

> 本仓库存在的意义是提供一整套，基于STM32H743IIT6型号MCU的，项目框架，方便后续其他项目能基于此来快速构建。

## 项目结构

```
project_name/
├── README.md           # 项目说明文档
├── LICENSE             # 项目许可证
├── .gitignore          # 忽略文件配置
├── doc/                # 文档目录
│ ├── mcu/              # 主控芯片资料
│ ├── board/            # 开发板资料
│ │ ├── xxx/            # 具体开发板型号
│ │ └── ...
│ └── api/              # 项目接口文档
├── hal/                # 硬件抽象层
│ │── cmsis/            # 芯片内核接口
│ │── inc/              # 接口声明
│ └── src/              # 接口实现
├── bsl/                # 板极支持层
│ │── drv/              # 板载设备驱动
│ └── rtos/             # 对接操作系统
├── osl/                # 操作系统层
├── mdl/                # 系统中间件层
│ └── lvgl/             # 轻量级图形库
└── svl/                # 系统业务层
```
